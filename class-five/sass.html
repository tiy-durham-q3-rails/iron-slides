What are CSS preprocessors?

CSS preprocessors take code written in the preprocessed language (LESS, Sass, Stylus) and then convert that code into CSS.

LESS

Most popularly know by being used by Twitter Bootstrap front end web framework.

Stylus

Sass

Syntactically Awesome StyleSheets

Sass is an extension of CSS that adds power and elegance to the basic language.

Variables

Variables begin with dollar signs, and are set like CSS properties and you can then refer to them in properties.

Nested Rules

Sass allows CSS rules to be nested within one another. The inner rule then only applies within the outer ruleâ€™s selector.

Operations

Sass has a handful of standard math operators like +, -, *, /, and % to use in our CSS.

Mixins

Mixins allow you to define styles that can be re-used throughout the stylesheet without needing to resort to non-semantic classes.

@extend directive

Using @extend lets you share a set of CSS properties from one selector to another and helps keep your Sass very DRY.

.sass VS .scss

.sass

	      	
// Variable
!primary-color= hotpink
 
// Mixin
=border-radius(!radius)
    -webkit-border-radius= !radius
    -moz-border-radius= !radius
    border-radius= !radius
 
.my-element
    color= !primary-color
    width= 100%
    overflow= hidden
 
.my-other-element
    +border-radius(5px)

.scss

	      	
// Variable
$primary-color: hotpink;
 
// Mixin
@mixin border-radius($radius) {
    -webkit-border-radius: $radius;
    -moz-border-radius: $radius;
    border-radius: $radius;
}
 
.my-element {
    color: $primary-color;
    width: 100%;
    overflow: hidden;
}
 
.my-other-element {
    @include border-radius(5px);
}

Compiling Sass

  1. First install Sass, directions on class website.
  2. Open Terminal.app
  3. cd to your Surf + Paddle folder.
  4. Type mv styles.css styles.scss
  5. Type: sass --watch styles.scss:styles.css
  6. Start getting Sassy!
