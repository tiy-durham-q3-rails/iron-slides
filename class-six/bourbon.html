Bourbon

What is Bourbon?

A simple and lightweight mixin library for Sass. 

          
// Bourbon mixin with Sass
section {
  @include linear-gradient(to top, red, orange);
}

/* Compiled CSS  */
section {
  background-color: red;
  background-image: -webkit-linear-gradient
                              (bottom, red, orange);
  background-image: 
                    linear-gradient(to top, red, orange);
}
          
        
          
// Bourbon mixin with Sass
h1 {
  font-family: $helvetica;
  font-size: golden-ratio(14px,  1);
}


/* Compiled CSS  */
h1 {
  font-family: "Helvetica Neue", 
                        Helvetica, Arial, sans-serif;
  font-size: 23px;
}

Bourbon VS Compass

  • Compass requires you to include a config.rb file to configure your Sass input/output.
  • Bourbons is simply a set of mixins for us to leverage.
  • Compass uses Ruby extensions while Bourbon relies exclusively on Sass.

Installing Bourbon

  1. Open Terminal.app
  2. Type: gem install bourbon
  3. cd into your Surf + Paddle styles directory.
  4. Type: bourbon install

Adding Bourbon to your SCSS file

  1. Open your .scss file.
  2. Add @import 'bourbon/bourbon'; to the beginning of your file.
  3. Enjoy the Bourbon goodness!

Bourbon documentation

There are lots of tips and tricks to use when using Bourbon. Check out their documentation located here: http://bourbon.io/docs/
