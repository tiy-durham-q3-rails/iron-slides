What is npm?

Node Package Manager (npm) installs, publishes and manages node programs through command line.

            
          npm install name-of-package

What is gulp.js?

A stream builder written in Javascript.

What do you mean by streams?

A stream represents an object that can either be the source of data or it can be the target of data.

"We should have some ways of connecting programs like garden hose--screw in another segment when it becomes necessary to massage data in another way. This is the way of I/O also."

Want to learn more about streams?

Check out the streams handbook!

gulp.js VS Grunt.js

  • gulp utilizes nodes built-in streams module to implement task, while Grunt does not.
  • Grunt causes you to write more code. (Input + output values need to be defined.)
  • gulp.js is newer, but building momentum.
