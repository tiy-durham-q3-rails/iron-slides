What is Javascript?

A client side scripting language that is primarily implemented as part of a web browser in order to provide enhanced user interfaces and dynamic websites.

Brief History

  • 1995: At Netscape, Brendan Eich created "LiveScript", which gets renamed to "JavaScript".
  • 1996: Microsoft releases "JScript", a port, for IE3.
  • 1997: JavaScript was standardized in the "ECMAScript" spec.
  • 2005: "AJAX" was coined, and the web 2.0 age begins.
  • 2006: jQuery 1.0 was released.
  • 2010: Node.JS was released by Joyent.
  • 2012: ECMAScript Harmony spec nearly finalized.

Variables

Variables can used to store values, strings or numbers.

            
            var x;
            x = 5;
            console.log(x);

Re-assigning values to an existing variable value.

            
            x = 50;
            console.log(x);
Variables are one of the basic building blocks of Javascript. Just like 'x' in algebra, a variable in programming is a named container for a value that can change. Variables have to be 'declared' once before being used, using the 'var' keyword and the name of the variable. A variable declaration is a simple statement, so it ends with a semi-colon. Assigning an initial value to a variable is called 'initialising'. You can declare and initialise in one go, or seperately. If its not assigned, it's undefined.

Variables Names

  • Begin with letters, $ or _
  • Only contain letters, numbers, $ and _
  • Case sensitive
  • Avoid reserved words
  • Choose clarity and meaning
  • Prefer camelCase for multipleWords (instead of under_score)
  • Pick a naming convention and stick with it

var numPeople, $mainHeader, _num, _Num;

Data Types

Variables can store different "types" of data.

  • Numbers
  • Strings
  • Booleans
  • Undefined
  • Null
Variables can be populated with any one of the 'data types' available in Javascript. These are the simple ones we'll be working with today... There are also data types that cater for situations where there is no value undefined and null. Loose typing- you don't tell JS what type of data type it is, it guesses. You can use typeof to see what it thinks.

Numbers

A number can be a whole or floating point value.

            
            var year = 2014;
            var pi = 3.14;

Strings

An immutable string of characters.

            
      var message = 'I am developer!';
      var book = "Dragon's Lair";

Booleans

Represents logical values which result in true or false.

            
          var chocolate = true;
          var kale = false;

Undefined

Represents a value that hasn't been defined.

            
          var notDefinedYet;

Null

Represents an explicitly empty value.

            
          var latte = null;

Functions

Functions are re-usable collections of statements.

            
    function functionName(parameters) {
      Statements
    }
            
            
    function todayDate() {
      console.log('June 11th');
    }
            
            
    todayDate();

Alternate Function Declaration Syntax

Functions are really just variables that have code inside, rather than strings or numbers.

var sum = function(a, b) {
    return a + b;
};

But really, functions are Objects, like most everything in JavaScript.

var sum = new Function('a', 'b', 'return a + b');

Arguments

In JavaScript, we have named arguments, but we can't refer to them by name during the function call.

            
      function sum(arg1, arg2) {
          return arg1 + arg2;
      }

      sum(88, 44);

Return Values

The return keyword returns a value to whoever calls the function (and exits the function).

            
function sum(num1, num2) {
  var result = num1 + num2;
  return result; // After won't be executed
}

Operators

  • Assignment: assigns value on left based on right value.
  • Comparison: Compares each value with a boolean result.
  • Arithmetic: takes numerical values and returns a single numerical value.
  • Logical: typically are boolean values using and, or, not.
  • String: concatenation of string values.
  • Bitwise: takes values as a set of 32 bits, rather than as decimal, hexadecimal, or octal numbers
  • Special: unique to Javascript operators (Listed here.)

  console.log('x is a positive value.');

while loop

The while loop tells JS to repeat statements until a condition is true.

          while (expression) {
            // statements to repeat
          }
          
          var x = 0;
          while (x < 5) {
            console.log(x);
            x = x + 1;
          }

for loop

The for loop is another way of repeating statements, more specialized than while.

for (initialize; condition; update) {
  // statements to repeat
}

for (var i = 0; i < 5; i = i + 1) {
  console.log(i);
}
