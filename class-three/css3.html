CSS3

What is CSS3?

CSS3 latest evolution of the Cascading Style Sheets language.

  • Aims at extending CSS2.
  • Experimental parts are vendor prefixed (e.g. -moz, -webkit).

border-radius

CSS property that allows us to define how rounded border corners are.

box-shadow

Allows casting a drop shadow from the frame of almost any element.

box-shadow: inset offset-x offset-y blur-radius spread-radius color;


box-shadow: inset 3px 3px 1em #0066CC, -1em 0 0.4em #CCCCCC;

gradient

A CSS data type that denotes a CSS <image> made of a progressive transition between two or more colors

linear-gradient

The CSS linear-gradient() function creates an which represents a linear gradient of color

A rainbow made from a gradient

background: linear-gradient(to right, red, orange, yellow, green, blue, indigo, violet);

radial-gradient

The CSS radial-gradient() function creates an which represents a gradient of colors radiating from an origin, the center of the gradient.

Bottom/left gradient

background-image: radial-gradient(circle at bottom left, yellow, #f06d06);
